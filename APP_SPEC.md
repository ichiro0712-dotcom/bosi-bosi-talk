## アプリ概要

**BOSHI×BOSHI Talk** は、カップル専用のコミュニケーションWebアプリです。
チャット、共有メモ、記念日管理、AIキャラクター「もち」との会話など、
2人の生活をサポートする機能を備えたPWA（Progressive Web App）です。

- **利用者**: ミルク（user_a）とメリー（user_b）の2人専用
- **技術構成**: Next.js + Supabase + Google Gemini AI
- **PWA対応**: スマホのホーム画面にインストール可能

## 画面構成

### ナビゲーション（共通）
- **PC**: 左サイドバー
- **スマホ**: 画面下部の固定メニュー（紫色 #9370db）

メニュー項目:
1. **ホーム** (`/`) — トップ画面
2. **チャット** (`/chat`) — メッセージ画面
3. **アルバム** — 外部Googleフォトへのリンク
4. **メモ** (`/memos`) — 共有メモ
5. **⋯（3点メニュー）** — 以下のサブメニュー:
   - 通知テスト — push通知の動作確認
   - 記念日設定 — 記念日の登録・編集
   - リマインダー — 定期通知の設定
   - もち設定 — AIキャラのプロンプト編集
   - アプリDL — PWAインストール
   - 切替 — プロフィール変更（ログアウト）

## 1. ホーム画面（トップページ）

**パス**: `/`

### 機能
- **記念日カウンター**: つきあった日からの経過日数、次の記念日までの日数を表示
- **背景画像**: 何もない場所を長押し（800ms）すると背景画像を変更可能
- **記念日設定**: カウンター部分を長押しすると記念日設定モーダルが開く

### 記念日設定モーダル
- メインの記念日（1つ）: トップのカウンターに反映
- その他の記念日（複数追加可能）: タイトルと日付を自由に設定
- 変更するともちがチャットに通知メッセージを投稿

---

## 2. チャット画面

**パス**: `/chat`

### 基本機能
- テキストメッセージの送受信
- 画像の送信（カメラ撮影 or アルバムから選択）
- リアルタイム同期（相手の送信が即座に表示される）
- 送信中インジケーター（↗マーク）
- 日付区切り表示
- メッセージのグルーピング（同じ人が連続で送ると吹き出しがつながる）
- リンクの自動検出とクリック対応

### 吹き出しの色
| 送信者 | 色 |
|--------|-----|
| 自分（通常メッセージ） | ピンク（#f0c8f7） |
| 相手（通常メッセージ） | 白 |
| もち or もち宛 | グレー（#e2e8f0） |

### スタンプ機能
- **既存スタンプ**: 64種類のカスタムスタンプを選んで送信
- **オリジナルスタンプ作成**: 写真をアップロード → AIが背景を自動除去 → テキスト追加 → スタンプとして保存・送信

### もちモード（AI会話）
1. `+` メニューから「もちと話す」を選択
2. 入力欄の上に「もちモード中」インジケーターが表示
3. メッセージを送信するとGemini AIが返答を生成
4. もちが考え中は `...` のタイピングアニメーション表示
5. 返答が届くとチャットに表示 + push通知

### push通知
- 通知未許可の場合、画面上部に「許可する」バーを表示
- 相手からメッセージが届くとスマホに通知
- 通知は「◯件の新着メッセージ」としてまとまる
- 通知タップでアプリにフォーカス

## 3. メモ画面

**パス**: `/memos`

### 機能
- 共有メモの一覧表示
- 新規メモ作成（＋ボタン）
- タイトルと本文の編集（自動保存、1秒後に反映）
- メモの削除（確認ダイアログあり）
- リアルタイム同期（相手が編集したメモも即反映）
- 最終更新日時の表示

## 4. もち（AIキャラクター）

### 概要
もちはアプリ内に住むAIキャラクターです。
チャットの中で2人をサポートし、会話を盛り上げます。

### もちが登場する場面
1. **チャットでの対話**: もちモードで話しかけると返答
2. **記念日変更通知**: 記念日設定を変更するとチャットに自動投稿
3. **背景画像変更通知**: トップ画面の背景を変更するとチャットに自動投稿
4. **リマインダー**: 設定した時刻にもちの名前で定期メッセージを投稿

### もちの設定
3点メニュー → 「もち設定」からシステムプロンプト（性格・口調・知識）を自由に編集可能。

---

## 5. リマインダー機能

### 概要
3点メニュー → 「リマインダー」から設定。
設定した時刻にもちがチャットにメッセージを自動投稿し、push通知を送ります。

### スケジュール種別
| 種別 | 説明 | 例 |
|------|------|-----|
| 1回だけ | 指定日時に1回実行 | 4月10日 18:00 |
| 毎日 | 毎日同じ時刻 | 毎日 9:00 |
| 毎週 | 毎週同じ曜日・時刻 | 毎週月曜 8:00 |
| 毎月（日付） | 毎月同じ日 | 毎月15日 |
| 毎月（第N週） | 毎月第N曜日 | 毎月第2月曜日 |

### 操作
- ON/OFF切り替え
- 削除
- 作成者の表示（ミルク/メリー）

## 6. push通知

### 仕組み
- Web Push API + Service Worker
- VAPID鍵による認証
- Vercelサーバーから全登録端末に配信

### 通知が届くタイミング
- 相手がチャットでメッセージを送信したとき（自分の送信では通知されない）
- もちがチャットに投稿したとき
- リマインダーが実行されたとき

### 通知の表示
- アプリアイコンとバッジ表示
- 振動パターンあり
- 複数件はまとめて「○件の新着メッセージ」
- ユーザーが消すまでホーム画面に残る

## 7. PWA（アプリインストール）

### インストール方法
- **Android**: 3点メニュー → 「アプリDL」またはブラウザメニューから
- **iPhone**: Safariで「共有」→「ホーム画面に追加」

### PWA機能
- ホーム画面にアプリアイコンが追加される
- フルスクリーン（standalone）表示
- push通知の受信
- オフラインでもアプリが開ける（データ取得にはネット必要）

## 操作ガイド（FAQ）

### Q: プロフィールを変更したい
A: 3点メニュー → 「切替」をタップすると、プロフィール選択画面に戻ります。

### Q: 通知が届かない
A: 以下を確認してください:
1. 3点メニュー → 「通知テスト」で確認
2. スマホの設定 → アプリ → BOSHI×BOSHI Talk → 通知がONか確認
3. バッテリー最適化から除外されているか確認

### Q: もちが反応しない
A: もちモード（+メニュー → もちと話す）がONになっているか確認してください。
もちモード中は入力欄の上に「もちモード中」と表示されます。

### Q: スタンプを自分で作りたい
A: +メニュー → 「スタンプ作成」→ 写真を選択 → 背景が自動で消える → テキストを追加 → 保存

### Q: 記念日を設定/変更したい
A: ホーム画面のカウンター部分を長押し、または3点メニュー → 「記念日設定」

### Q: リマインダーを設定したい
A: 3点メニュー → 「リマインダー」→ 「＋新しいリマインダー」

### Q: 背景画像を変更したい
A: ホーム画面で何もない場所を長押し（約1秒）すると画像選択が開きます。
