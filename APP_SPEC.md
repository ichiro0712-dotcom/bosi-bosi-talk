# BOSHI×BOSHI Talk — アプリ仕様書

> 最終更新: 2026-04-06
> このドキュメントはアプリの全機能を網羅した仕様書です。
> もちAIのSystem Promptにも使用されます。

---

## アプリ概要

**BOSHI×BOSHI Talk** は、カップル専用のコミュニケーションWebアプリです。
チャット、共有メモ、TODO管理、記念日管理、AIキャラクター「もち」との会話など、
2人の生活をサポートする機能を備えたPWA（Progressive Web App）です。

- **利用者**: ミルク（user_a）とメリー（user_b）の2人専用
- **技術構成**: Next.js + Supabase + Google Gemini AI
- **PWA対応**: スマホのホーム画面にインストール可能

---

## ユーザー

| 名前 | ID | 説明 |
|------|-----|------|
| ミルク | user_a | ユーザーA |
| メリー | user_b | ユーザーB |
| もち | mochi | AIキャラクター（ボット） |

アプリ起動時にプロフィール選択画面で「ミルク」か「メリー」を選ぶ。

## 画面構成

### ナビゲーション（共通）
- **PC**: 左サイドバー
- **スマホ**: 画面下部の固定メニュー（紫色 #9370db）

メニュー項目:
1. **ホーム** (`/`) — トップ画面
2. **チャット** (`/chat`) — メッセージ画面
3. **アルバム** — 外部Googleフォトへのリンク
4. **TODO** (`/todos`) — タスク・リマインダー管理
5. **メモ** (`/memos`) — 共有メモ
6. **⋯（3点メニュー）** — 以下のサブメニュー:
   - 通知テスト — push通知の動作確認
   - 記念日設定 — 記念日の登録・編集
   - もちAI設定 (`/mochi-settings`) — AI設定ページ
   - アプリDL — PWAインストール
   - 切替 — プロフィール変更（ログアウト）

## 1. ホーム画面（トップページ）

**パス**: `/`

### 機能
- **記念日カウンター**: つきあった日からの経過日数、次の記念日までの日数を表示
- **背景画像**: 何もない場所を長押し（800ms）すると背景画像を変更可能
- **記念日設定**: カウンター部分を長押しすると記念日設定モーダルが開く
- **今週のTODO**: フッターの上に最大3件表示、タップでTODOページへ

### 記念日設定モーダル
- メインの記念日（1つ）: トップのカウンターに反映
- その他の記念日（複数追加可能）: タイトルと日付を自由に設定
- 変更するともちがチャットに通知メッセージを投稿

## 2. チャット画面

**パス**: `/chat`

### 基本機能
- テキストメッセージの送受信
- 画像の送信（カメラ撮影 or アルバムから選択）
- リアルタイム同期（相手の送信が即座に表示される）
- 日付区切り表示
- メッセージのグルーピング（同じ人が連続で送ると吹き出しがつながる）
- リンクの自動検出とクリック対応

### メッセージステータス（LINE準拠）
| ステータス | 表示 | 条件 |
|-----------|------|------|
| 送信中 | ↗ | 送信ボタン押下〜サーバー到達 |
| 未読 | 時刻のみ | サーバー到達〜相手が画面を開くまで |
| 既読 | 紫色「既読」+ 時刻 | 相手がチャット画面を開いた時 |
| エラー | 赤い！マーク | 送信失敗時。再送・削除ボタン付き |

### 吹き出しの色
| 送信者 | 色 |
|--------|-----|
| 自分（通常メッセージ） | ピンク（#f0c8f7） |
| 相手（通常メッセージ） | 白 |
| もち or もち宛 | グレー（#e2e8f0） |

### 長押しメニュー（LINE準拠）
メッセージを500ms長押し or 右クリックで表示。

| 機能 | 説明 |
|------|------|
| **リプライ** | 引用返信。吹き出し上部にプレビュー表示。タップで元メッセージにジャンプ+ハイライト |
| **コピー** | テキストをクリップボードにコピー |
| **アナウンス** | チャット上部にピン留め（最大5件）。タップでジャンプ。展開リストあり |
| **送信取り消し** | 自分のメッセージのみ、24時間以内。「取り消しました」テキスト残る |
| **削除** | 自分の画面からのみ消える。痕跡なし。時間制限なし |

### スタンプ機能
- **既存スタンプ**: 64種類のカスタムスタンプを選んで送信
- **オリジナルスタンプ作成**: 写真をアップロード → AIが背景を自動除去 → テキスト追加 → スタンプとして保存・送信

### もちモード（AI会話）
1. `+` メニューから「もち」を選択
2. 入力欄の上に「もちモード中」インジケーターが表示
3. メッセージを送信するとGemini AIが返答を生成
4. もちが考え中は `...` のタイピングアニメーション表示
5. 返答が届くとチャットに表示 + push通知

## 3. メモ画面

**パス**: `/memos`

### 機能
- 共有メモの一覧表示

## 4. TODO / リマインダー

**パス**: `/todos`

### タブ構成
- **Task**: 進行中のタスク一覧
- **リマインダー**: 定期通知の設定・管理
- **完了Task**: 完了したタスクのアーカイブ

### タスク機能
- 親タスクと小タスクの階層構造
- 各タスクに担当者（ミルク / メリー / 2人）を設定
- 期限の設定。期限バッジ: 今日中（オレンジ）、今週中（紫）、期限超過（赤）
- ステータス管理: まだ / 順調 / トラブル / 遅れてる / 止まってる / 完了
- 内容メモ（description）フィールド
- プログレスバー（親タスクに子タスクの進捗表示）
- タップで編集モード切替

### リマインダー機能
設定した時刻にもちがチャットにメッセージを自動投稿し、push通知を送る。

| 種別 | 説明 |
|------|------|
| 1回だけ | 指定日時に1回実行 |
| 毎日 | 毎日同じ時刻 |
| 毎週 | 毎週同じ曜日・時刻 |
| 毎月（日付） | 毎月同じ日 |
| 毎月（第N週） | 毎月第N曜日 |

### HOME画面連携
ホーム画面に「今週のTODO」ウィジェットが表示（最大3件、超過分は件数表示）

### もちとの連携
- 「○○やらなきゃ」→ もちがTODOに自動追加
- 「○○終わった」→ もちがステータスを完了に変更
- 「○○いらない」→ もちがタスク削除
- 「毎朝9時にリマインドして」→ もちがリマインダー追加
- 「○○のリマインダー止めて」→ もちがON/OFF切替
- 「やること教えて」→ もちが一覧で教える

## 5. もち（AIキャラクター）

### 概要
もちはアプリ内に住むAIキャラクター。Gemini 2.5 Flashで動作。
3層メモリアーキテクチャ + Function Callingで自律的に学習・記憶。

### メモリ構造
| 層 | 内容 |
|----|------|
| Layer 0 | アプリ仕様書（このドキュメント） |
| Layer 1 | ユーザープロファイル（カテゴリ別）+ 関係性Vibe |
| Layer 2 | 過去の会話サマリー（50件ごとに自動圧縮） |
| Layer 3 | 直近30件の生チャットログ |

### ユーザープロファイル（カテゴリ別）
もちが記憶する情報は7カテゴリに整理される:
- 基本情報 / 性格・価値観 / 事業・仕事 / 健康・生活 / お金 / 趣味・興味 / その他

### Function Calling（自律操作）
もちは会話中に以下のツールを自動実行:
- ユーザー情報の記憶（カテゴリ別に追記）
- 関係性Vibeの更新
- タスクの追加・ステータス変更・削除
- リマインダーの追加・ON/OFF・削除

### プロアクティブ通知（Cron）
- 毎朝9:00: 今日期限のタスクがあれば「大丈夫？」と声かけ
- 毎週月曜10:00: 今週のタスク一覧を通知

### もちが登場する場面
1. チャットでの対話（もちモード）
2. 記念日変更通知
3. 背景画像変更通知
4. リマインダーの自動投稿
5. プロアクティブなタスク通知

### もちの設定ページ (`/mochi-settings`)
5つのタブ:
- **キャラ設定**: システムプロンプト（性格・口調）の編集
- **ユーザー情報**: 7カテゴリの自由記述テキストエリア
- **関係性**: 2人の雰囲気（ラブラブ/普通/ピリピリ等）の確認・変更
- **会話サマリー**: 自動生成された要約の閲覧
- **記憶ログ**: もちの自律記憶更新の履歴

## 6. push通知

### 仕組み
- Web Push API + Service Worker
- VAPID鍵による認証
- Vercelサーバーから全登録端末に配信

### 通知が届くタイミング
- 相手がチャットでメッセージを送信したとき（自分の送信では通知されない）
- もちがチャットに投稿したとき
- リマインダーが実行されたとき
- プロアクティブ通知（朝のタスク確認、週次タスクサマリー）

### 通知の表示
- アプリアイコンとバッジ表示（もちの通知はもちアイコン）
- 振動パターンあり
- 複数件はまとめて「○件の新着メッセージ」
- ユーザーが消すまでホーム画面に残る

## 7. PWA（アプリインストール）

### インストール方法
- **Android**: 3点メニュー → 「アプリDL」またはブラウザメニューから
- **iPhone**: Safariで「共有」→「ホーム画面に追加」

### PWA機能
- ホーム画面にアプリアイコンが追加される
- フルスクリーン（standalone）表示
- push通知の受信
- オフラインでもアプリが開ける（データ取得にはネット必要）

## 操作ガイド（FAQ）

### Q: プロフィールを変更したい
A: 3点メニュー → 「切替」をタップ。

### Q: 通知が届かない
A: 3点メニュー → 「通知テスト」で確認。スマホの設定でアプリの通知がONか、バッテリー最適化から除外されているか確認。

### Q: もちが反応しない
A: もちモード（+メニュー → もち）がONか確認。入力欄の上に「もちモード中」と表示される。

### Q: スタンプを自分で作りたい
A: +メニュー → 「作成」→ 写真を選択 → 背景が自動で消える → テキスト追加 → 保存。

### Q: 記念日を設定/変更したい
A: ホーム画面のカウンター部分を長押し、または3点メニュー → 「記念日設定」。

### Q: リマインダーを設定したい
A: TODOページ → 「リマインダー」タブ → 右上の「＋リマインダー」。

### Q: 背景画像を変更したい
A: ホーム画面で何もない場所を長押し（約1秒）すると画像選択が開く。

### Q: メッセージを取り消したい
A: メッセージを長押し → 「送信取り消し」（24時間以内のみ）。

### Q: メッセージを引用して返信したい
A: メッセージを長押し → 「リプライ」。

### Q: 重要なメッセージをピン留めしたい
A: メッセージを長押し → 「アナウンス」。チャット上部に固定表示される。

### Q: もちにタスクを追加してもらいたい
A: もちモードで「○○をTODOに入れて」と伝える。

### Q: もちの性格を変えたい
A: 3点メニュー → 「もちAI設定」→ キャラ設定タブでシステムプロンプトを編集。
